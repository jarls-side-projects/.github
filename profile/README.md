# Jarls Side Projects

A GitHub organization for personal side projects built and maintained by [@jarlah](https://github.com/jarlah).

## What's here

This org is a home for open source experiments and hobby projects, primarily written in [Elixir](https://elixir-lang.org/). The projects here reflect a mix of practical tools built for real use and exploratory work done for the love of building.

### Active projects

- **[elixir-wenche](https://github.com/jarls-side-projects/elixir-wenche)** — An open source Elixir library. Public and open to contributions.
- An accounting tool for small businesses is currently in development.

## Tech stack

Most projects here are built with:

- [Elixir](https://elixir-lang.org/) / [Phoenix](https://www.phoenixframework.org/)
- [Ash Framework](https://ash-hq.org/)
- [Phoenix LiveView](https://hexdocs.pm/phoenix_live_view/)

## Contributing

Public repositories are open to issues and pull requests. Feel free to open an issue if you find a bug or have a suggestion.
